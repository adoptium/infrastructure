⚠️ Cloud Configuration Not Available

Cloud capacity reporting is disabled because the clouds.xml configuration file is not present.

To enable cloud capacity reporting:

  1. On your Jenkins server, run the extraction script:
    cd /path/to/jenkins-capacity-report/tools
    ./extract_clouds_config.sh -o ../data/clouds.xml.live
  2. Copy the generated clouds.xml file to this application directory
  3. Update the CLOUD_CONFIG_FILE variable in your .env file:
    CLOUD_CONFIG_FILE=./data/clouds.xml.live
  4. Restart the web application

Note: The extraction script must be run on the Jenkins controller server where the main config.xml file is located.

For detailed instructions, see the README.md file.
